# DeepSeek Notes

基于 DeepSeek 模型的本地 AI 知识管理工具。

## 项目说明

- 创建时间：2026-06-08
- 技术栈：Electron + Vue 3 + TypeScript + SQLite
- 核心理念：「对话即笔记，笔记即上下文」

## 文档索引

- `PRD.md` — 产品需求文档（完整功能定义）
- `TECH_STACK.md` — 技术栈与架构设计（含数据库 Schema）
- `DEV_PLAN.md` — 7 个 Sprint 开发计划

## 功能概览

三个 Tab：
1. **AI 笔记** — DeepSeek 对话，支持对话历史、笔记存入、笔记引用、联网搜索
2. **笔记本** — 笔记管理，分类/标签/Markdown 5主题/全文搜索
3. **个人中心** — Dashboard、API Key 配置、导入导出
